Aby uruchomić program należy:

Utworzyć folder `build`
```
  mkdir build && cd build
```

Skompilować projekt:

Wersja linuksowa:
```
  cmake .. && make
```
Wersja windowsowa
```
  cmake ..
  cmake --build
```

Uruchomienie programu wykonywalnego
```
  ./main <plik>
```

lub testów
```
  ./SATtest
```
